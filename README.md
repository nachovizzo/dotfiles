# Nacho dotfiles

Work in progress (almost finished) transition repository using yadm as tool

## Dependencies

- `curl`
- `git`
- root access

## Installation

```sh
sh -c "$(curl -fsSL https://raw.githubusercontent.com/nachovizzo/dotfiles/main/.config/yadm/install.sh)" "" --decrypt
```
